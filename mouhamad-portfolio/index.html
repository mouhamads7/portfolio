Agroleaf

This website was designed during our final bachelor's project with three other students. It involved creating a platform for farmers to help them have a comprehensive and simple view of the progress of their crops. This project was proposed by a client, and we had to work closely with them

This project includes several features, including:

- Authentication and user creation

- Company creation with the addition of shareholders

- Editing user and company information

- Adding lots and divisions with the ability to visualize them on a map. Additionally, you can access the details of each lot to know its profitability by clicking on it

- Adding types of crops to each lot/division and performing calculations to generate projections for them

This application was built using the following technology stack:

- For the front-end: React, typescript, tailwind

- For the back-end: Golang, postgresql

- Development Environment: VScode, docker, vite

- Deployment: Digital ocean
